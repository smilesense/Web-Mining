Web Mining#

Web Mining adalah proses teknik Data Mining untuk secara otomatis menemukan dan mengekstrak informasi dari suatu halaman Web. Tujuan utama penambangan web adalah menemukan informasi yang berguna.

Penambangan web membantu meningkatkan kekuatan mesin pencari web dengan mengklasifikasikan dokumen web dan mengidentifikasi halaman web.

Ensemble Learning#

Ensemble Learning adalah proses di mana beberapa model, seperti pengklasifikasi atau ahli, secara strategis dihasilkan dan digabungkan untuk memecahkan kecerdasan komputasi tertentu.masalah. Ensemble Learning utamanya digunakan untuk meningkatkan (klasifikasi, prediksi, perkiraan fungsi, dll.) kinerja model, atau mengurangi kemungkinan pemilihan model yang buruk.

def process_comments(response_items, csv_output=False):
    
    for res in response_items:

        # loop through the replies
        if 'replies' in res.keys():
            for reply in res['replies']['comments']:
                comment = {}
                comment['videoId'] = reply['snippet']['videoId']
                comment['parentId'] = reply['snippet']['parentId']
                comment['commentId'] = reply['id']
                # comment.append('Label')
                comment['textDisplay'] = reply['snippet']['textDisplay']
                comment['Label'] = "None"
                # print (comment)
                comments.append(comment)
        else:
            comment = {}
            comment['videoId'] = res['snippet']['topLevelComment']['snippet']['videoId']
            comment['parentId'] = None
            comment['commentId'] = res['snippet']['topLevelComment']['id']
            # comment.append('Label')
            comment['textDisplay'] = res['snippet']['topLevelComment']['snippet']['textDisplay']
            comment['Label'] = "None"
            print(comment)
            
            comments.append(comment)

    if csv_output:
         make_csv(comments)
    
    print(f'Finished processing {len(comments)} comments.')
    return comments


def make_csv(comments, channelID=None):
    header = comments[0].keys()

    filename = f'comments_raw.csv'
   

    with open(filename, 'w', encoding='utf8', newline='') as f:
        writer = csv.DictWriter(f, fieldnames=header, extrasaction='ignore')
        writer.writeheader()
        writer.writerows(comments)

    api_service_name = "youtube"

    comments_list = []

    youtube = googleapiclient.discovery.build(

    request = youtube.commentThreads().list(
        part="id, replies, snippet",
        videoId=idvid
    )
    response = request.execute()
    comments_list.extend(process_comments(response['items']))

    # if there is nextPageToken, then keep calling the API
    while response.get('nextPageToken', None):
        request = youtube.commentThreads().list(
            part='id,replies,snippet',
            videoId=idvid,
            pageToken=response['nextPageToken']
        )
        response = request.execute()
        comments_list.extend(process_comments(response['items']))

    comments_list = list(unique_everseen(comments_list))

    print(f"Finished fetching comments for {idvid}. {len(comments_list)} comments found.")
    
    make_csv(comments_list, idvid)
    
    return comments_list

Case Folding#

Case folding merupakan tahap text preprocessing yang berguna untuk :

  • Mengubah huruf kapital menjadi huruf kecil

  • Mengapus tanda baca

  • Menghapus angka

  • Menghapus karakter kosong


Sebelum melakukan case folding diatas, kode ini juga melakukan penghapusan link dan mention yang ada pada data tweet.

Term Frequency (TF)#

Term Frequency merupakan metode yang digunakan untuk mengetahui seberapa sering suatu kata muncul dalam suatu teks. Semakin banyak frekuensi kemunculan dari kata tsb, semakin besar pula nanti nilainya.

listkata = []
for tweet in df['textDisplay']:

listkata_nodup = list(dict.fromkeys(listkata))
hasil_hitung = {}
for test1 in listkata_nodup:
    jumlah = 0
    for test2 in range(len(listkata)):
        if test1 == listkata[test2]:
            jumlah+=1
    hasil_hitung.update({'%s'%test1 : jumlah})

Stacking Classifier#

Stacking adalah teknik pembelajaran ensemble untuk menggabungkan beberapa model klasifikasi melalui meta-classifier. Model klasifikasi individu dilatih berdasarkan set pelatihan lengkap; kemudian, pengklasifikasi meta dipasang berdasarkan keluaran – fitur-meta – dari model klasifikasi individu dalam ansambel. Pengklasifikasi meta dapat dilatih pada label kelas yang diprediksi atau probabilitas dari ansambel.
